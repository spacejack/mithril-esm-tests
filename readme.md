# Mithril ESM bundler tests

## Install

    npm i

## Test webpack:

    npm run wp

## Test browserify:

    npm run br

## Test rollup:

    npm run ro

## Notes

The root level `tsconfig.json` is used by webpack.

The `src/tsconfig.json` is used by browserify.

The `tsconfig.rollup.json` is used by rollup.
